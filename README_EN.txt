* How to use this project
- downloads one set to document_root of apache or clones it, carry out.
- Make a cache/twig directory to the subordinate of documentroot/sample/(the compiled template of Twig is arranged here).
- If documentroot/sample/twig/test.php is accessed, the result of having processed sample/sample/twig/templates/test.html should be displayed.

- composer.phar
http://getcomposer.org/
The library installer of PHP.
Seemingly, a dependency will also be canceled in passing.

Installation of the composer itself
curl -sS https://getcomposer.org/installer | php
If curl is not contained, it will download from upper URL.

Usage describes a library to install by composer.json and
php composer.phar install
